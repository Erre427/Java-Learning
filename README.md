## 🛤️ Roadmap de Tecnologías a Seguir

A lo largo del curso, estaré cubriendo y subiendo proyectos sobre las siguientes tecnologías:

### 🔹 Java Core & Desktop
* **Java Standard Edition (Java SE):** Fundamentos y lógica de programación.
* **Desktop Applications:** Creación de interfaces gráficas.

### 🔹 Java Enterprise & Web
* **Java EE / Jakarta EE:** Desarrollo de aplicaciones empresariales.
* **Web:** Servlets, JSP y despliegue en servidores.

### 🔹 Persistencia y Datos
* **JPA (Java Persistence API):** Estándar de persistencia de datos.
* **Hibernate:** Framework de ORM avanzado.

### 🔹 Calidad y Testing
* **JUnit:** Pruebas unitarias para asegurar la robustez del código.

### 🔹 Ecosistema Spring & Arquitectura Modernas
* **Spring Boot:** Desarrollo rápido de aplicaciones productivas.
* **Spring Cloud:** Configuración y patrones para sistemas distribuidos.
* **Microservicios:** Arquitecturas escalables y desacopladas.
* **Full Stack:** Integración del backend con tecnologías frontend.

## 🛠️ Cómo ejecutar las prácticas

Si deseas clonar este repositorio y probar los ejercicios localmente, sigue estos pasos:

1. **Clonar el repositorio:**
   ```bash
   git clone [https://github.com/Erre427/TU_REPOSITORIO.git](https://github.com/Erre427/TU_REPOSITORIO.git)
2.  **Verificar la versión de Java:**
    
    Bash
    
    ```
    java -version
    
    ```
    
3.  **Compilar y ejecutar:** Navega hasta la carpeta del ejercicio y usa:
    
    Bash
    
    ```
    javac NombreDelEjercicio.java
    java NombreDelEjercicio
    ```
